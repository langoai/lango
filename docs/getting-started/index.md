# Getting Started

Get Lango up and running in minutes.

## Overview

Setting up Lango involves three steps:

1. **[Installation](installation.md)** -- Build the binary from source
2. **[Quick Start](quickstart.md)** -- Run the onboard wizard and start the server
3. **[Configuration](configuration.md)** -- Understand profiles and config management

## Requirements at a Glance

| Requirement | Minimum |
|---|---|
| Go | 1.25+ |
| CGO | Enabled (required for SQLite) |
| OS | Linux, macOS |
| AI Provider | At least one API key (OpenAI, Anthropic, Gemini, or Ollama) |

!!! tip

    If you just want to try Lango quickly, follow the [Quick Start](quickstart.md) guide -- it takes about 2 minutes.
